1. 아이디 유효성 검사
 - 필수입력 요소 
 - 영문만 입력 가능 ( 정규표현식 사용)

2. 비밀번호 유효성 검사
 - 필수입력 요소 
 - 숫자영문만 입력 가능 ( 정규표현식 사용)

3. 비밀번호 확인
 - 필수입력 요소 
 - 입력한 비밀번호와 일치해야함.

4. 이름
 - 필수입력 요소
 - 한글만 입력 가능(정규표현식 사용)

5. 생년월일
 - 필수입력 요소
 - 연도 4자리 고정, 일은 1~31까지만 가능

6. 성별
 - 필수입력 요소(반드시 하나라도 선택되어야함)

7. 이메일
 - 필수입력 요소

8. 연락처 
 - 필수입력 요소
 - 가운데 자리 최소 3자리 최대 4자리 숫자만
 - 마지막 자리 4자리 숫자만

* 각 모든 입력 양식은 입력 후 커서가 벗어났을 때 유효성 검사 후 각 입력양식 아래 checkRs 요소에 결과 문구를 안내한다.
* 회원가입 버튼 클릭시 모든 양식의 값이 유효한 경우만 데이터 전송된다.
